# Guide candidat pas a pas (version simple)

Ce guide est fait pour vous aider a passer le test tranquillement, meme si vous n'etes pas a l'aise avec l'informatique.

Prenez votre temps. Vous pouvez lire chaque etape une par une.

## 1. Avant de commencer

Vous avez besoin de :
- un ordinateur, une tablette ou un telephone,
- une connexion internet,
- le code d'acces au test (donne par votre animateur).

Conseil :
- installez-vous dans un endroit calme,
- gardez ce guide ouvert a cote.

## 2. Ouvrir le test

1. Ouvrez votre navigateur internet (Chrome, Edge, Firefox...).
2. Allez sur la page du test (l'animateur vous fournira le lien).
3. Vous voyez une case "Code d'acces".
4. Tapez le code donne.
5. Cliquez sur le bouton "Acceder au test".

Si un message "Code invalide" s'affiche :
- verifiez que vous avez bien tape le bon code,
- faites attention aux chiffres,
- demandez de l'aide a l'animateur si besoin.

## 3. Entrer votre nom et votre prenom

Avant les questions, vous devez remplir :
- votre nom,
- votre prenom.

Puis cliquez sur "Commencer le questionnaire".

Important :
- sans le nom et le prenom, le test ne peut pas commencer.

## 4. Repondre aux questions

Vous aurez plusieurs types de questions.

## 4.1 Questions a choix (QCM)

- Cliquez sur la reponse que vous pensez juste.
- Si vous changez d'avis, cliquez sur une autre reponse.

## 4.2 Questions avec texte libre

- Ecrivez votre reponse dans la grande zone de texte.
- Ecrivez avec des mots simples, c'est suffisant.

## 4.3 Question pictogrammes

- Regardez l'image.
- Ecrivez ce qu'elle signifie dans la case.

## 4.4 Aller a la question suivante

Vous pouvez :
- utiliser les fleches bas de page (precedent / suivant),
- ou cliquer sur un numero de question dans la navigation.

## 4.5 Verifier l'avancement

La barre de progression montre :
- ce que vous avez deja repondu,
- ce qu'il reste a faire.

Si des questions manquent, le test ne pourra pas etre envoye.

## 5. Signature du candidat

A la fin, vous devez signer.

1. Signez avec le doigt (telephone/tablette) ou la souris (ordinateur).
2. Si la signature n'est pas belle, cliquez sur "Effacer" et recommencez.
3. Cliquez sur "Utiliser cette signature".

Important :
- sans signature, l'envoi final est bloque.

## 6. Verification finale (tres important)

Quand tout est rempli :
1. Cliquez sur "Verification finale".
2. Un resume apparait avec :
   - votre nom et prenom,
   - toutes vos reponses.
3. Relisez calmement.

Ensuite, vous avez 2 choix :
- "Retour au questionnaire" pour corriger,
- "Confirmer et envoyer" pour terminer.

## 7. Apres l'envoi

Apres "Confirmer et envoyer" :
- vous arrivez sur la page des resultats,
- votre score s'affiche,
- vous pouvez telecharger le PDF.

## 8. Problemes frequents et solutions simples

## 8.1 "Je ne peux pas valider"

Verifiez :
- que toutes les questions sont remplies,
- que vous avez bien signe,
- que nom et prenom sont remplis.

## 8.2 "Je me suis trompe"

- Revenez a la question concernee,
- modifiez la reponse,
- revenez sur "Verification finale".

## 8.3 "La page ne repond pas"

- attendez 5 a 10 secondes,
- rechargez la page (touche F5 sur ordinateur ou fleche en forme de rond en haut a gauche),
- si besoin, demandez de l'aide.

## 8.4 "Je n'arrive pas a telecharger le PDF"

- verifiez votre connexion internet,
- cliquez de nouveau sur le bouton de telechargement,
- demandez a l'animateur si le probleme continue.

## 9. Conseils pour etre serein

- Prenez votre temps, il n'y a pas besoin d'aller vite.
- Lisez chaque question tranquillement.
- Si vous ne comprenez pas une question, demandez de l'aide.
- Le plus important est de repondre de facon sincere et complete.
